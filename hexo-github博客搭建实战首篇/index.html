安装 node.js
安装 git

安装

hexo npm install -g hexo-cli

初始化 hexo 文件夹

下载 next 主题 配置next

开启 分类和标签 页

创建新文章

开启网站

生成静态页

发布到github并设置自定义域名

购买域名 设置 A记录（192.30.252.153 or 192.30.252.154） 配置CNAME指向该A记录

创建github仓库 xxxx.github.io

安装 git-deploy

hexo _config.yml 配置 deploy
